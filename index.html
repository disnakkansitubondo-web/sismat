Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Tambah Surat
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan link Google Drive sudah benar
  4. Klik tombol Simpan Surat

Anda dapat mencari dan memfilter data surat dengan cara:

  • Gunakan kotak pencarian untuk mencari berdasarkan nomor atau perihal surat
  • Gunakan filter jenis, status, dan tahun untuk menyaring data
  • Klik tombol Terapkan Filter untuk menerapkan filter
  • Klik tombol Reset untuk menghapus semua filter

Untuk membuat laporan:

  1. Buka menu Laporan
  2. Pilih tahun, bulan, dan jenis surat yang ingin dilaporkan
  3. Sistem akan menampilkan grafik dan tabel data sesuai filter
  4. Klik tombol Cetak Laporan untuk mencetak laporan

Backup Data:

  1. Buka menu Backup & Restore
  2. Klik tombol Backup Data Sekarang
  3. File backup akan diunduh secara otomatis

Restore Data:

  1. Buka menu Backup & Restore
  2. Klik Pilih File dan pilih file backup sebelumnya
  3. Klik tombol Restore Data
  4. Data akan diganti dengan data dari file backup
Pemecahan Masalah

Pastikan semua field wajib telah diisi dan koneksi internet stabil. Jika masalah berlanjut, coba:

  • Refresh halaman
  • Clear cache browser
  • Hubungi developer

Jika laporan tidak muncul, pastikan:

  • Filter yang diterapkan benar
  • Ada data untuk periode yang dipilih
  • Tidak ada masalah dengan grafik
